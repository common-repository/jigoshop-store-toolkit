== Description ==

Store Toolkit includes a growing set of commonly-used Jigoshop administration tools aimed at web developers and store maintainers.

**Features**

1. Nuke support for clearing Jigoshop store records

* Products
* Product Images
* Produt Tags
* Product Categories
* Orders
* Coupons

2. Nuke support for clearing WordPress records

* Posts
* Post Categories
* Post Tags
* Links
* Comments

3. Jigoshop maintainence/debugging tools

* Re-link existing Sales from pre-registered Users
* All in One SEO Pack integration
* File Download Authorization widget on Orders screen
* Order Meta widget on Orders screen
* Product Meta widget on Products screen
* Jigoshop Dashboard widgets on the WordPress Dashboard screen

If you find yourself in the situation where you need to start over with a fresh installation of Jigoshop then a 'nuke' will do the job.

For more information visit: http://www.visser.com.au/jigoshop/

== Installation ==

1. Upload the folder 'jigoshop-store-toolkit' to the '/wp-content/plugins/' directory
2. Activate 'Jigoshop - Store Toolkit' through the 'Plugins' menu in WordPress

That's it!

== Usage ==

1. Open Jigoshop > Store Toolkit
2. Select which Jigoshop details you would like to remove and click Nuke

Done!
